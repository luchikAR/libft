# :white_check_mark: Libft (project 2020) 
## Описание 
Основная задача - написать собственную библиотеку функций для дальнейшего использования в проектах на C.
Функции должны быть реализованы таким же образом, как и исходные функции из стандартной библиотеки.
Однако все функции должны быть защищены от ввода неверных данных, утечек памяти, ложной сегментации и так далее.

## Запуск
Основные команды:
```
make (make all) - compilation of 1 part and 2 part functions  
make bonus - compilation of 1 part, 2 part and bonus part functions  
make clean - clearing *.o files  
make fclean - clearing *. o files and libraries  
```
